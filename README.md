# Calculator-C
Calculadora feita em C

Esta calculadora realiza as seguintes operações matemáticas:

- Soma
- Subtração
- Multiplicação
- Divisão

## Linux ##
Para rodar esta calculadora no linux, certifique-se que tenha instalado o gcc.

- Soma:
```
gcc src/soma.c -o soma
chmod +x ./soma
./soma
```
